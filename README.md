# 💣 Сапёр (Minesweeper) с мировым рейтингом

Классическая игра «Сапёр» на чистом JavaScript. Поле 16x16, 40 мин.

## 🕹️ Демо
[Играть онлайн](https://mines-weeper-web.netlify.app/)

## 🚀 Особенности
- 🎮 Поле 16x16, 40 мин (классический режим)
- 💥 Анимация взрыва (система частиц) и тряска экрана
- ✨ Свечение клеток при победе
- 🏆 Глобальная таблица рекордов через MockAPI
- ⏱️ Точный таймер и счётчик оставшихся мин
- 🎵 Звуковое сопровождение (установка флага, победа, поражение, фоновая музыка)

## 🛠️ Технологии
- HTML5 Canvas
- CSS3 (Flexbox, Glassmorphism, адаптив)
- JavaScript (ES6+): рекурсия (Flood Fill), Fetch API, Async/Await, система частиц

## 🎮 Как играть
- **Левый клик** — открыть ячейку
- **Правый клик** — поставить/убрать флажок (🚩)
- Цель: открыть все ячейки без мин как можно быстрее

## 📂 Структура
- `index.html` — главный файл
- `style.css` — стили
- `app.js` — игровая логика
- `music/` — звуки
- `credits.md` — авторские права на ассеты
